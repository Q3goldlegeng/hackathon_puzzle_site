# Contributing Guide

## Branch Rules
- main: stable, demo only
- dev: integration branch
- feature/*: new features

## Workflow
1. Create a feature branch from dev
2. Commit your changes
3. Open a Pull Request to dev
4. Wait for review

## Commit Message
Use simple English:
- add puzzle 1
- fix verify logic

# 貢獻指南

## 分支規則

- main：穩定版，僅供演示

- dev：整合分支

- feature/*：新功能分支

## 工作流程

1. 從 dev 分支建立功能分支

2. 提交更改

3. 向 dev 分支提交 Pull Request

4. 等待審核

## 提交訊息

請使用簡潔的英語：

- 新增謎題 1

- 修復驗證邏輯
